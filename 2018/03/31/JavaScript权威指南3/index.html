JavaScript权威指南3

跳转


  break语句是跳转到循环或者其他语句的结束;continue语句是中止本次循环的执行并开始下一次循环的执行;return语句让解释器跳出函数体的执行,并提供本次调用的返回值;throw语句触发或者”抛出”一个异常.

标签语句


  通过给语句定义标签,就可以在程序的任何地方通过标签名引用这条语句.

break语句


  单独使用break语句的作用是立即退出最内层的循环或switch语句.

continue语句


  continue语句和break语句类似,但它不是退出循环,而是转而执行下一次循环.

不赞成使用with语句.
